아래 순서대로 정리해 두었고, **이미지 제공이 끝나면 "끝났어"라고 하면 그때 코드에 반영**합니다.

## 슬라이드 1 (표지) 채울 내용

- **{광고주명}** → 실제 광고주명
- **N.square** → 브랜드/서비스명 (템플릿 그대로 또는 설정값)
- **광고배너 게재보고서** → 고정 문구
- **{년.월.일}** → 보고 일자 (YYYY-MM-DD 형식 등)
- 하단: kt nasmedia 로고·저작권 문구 (템플릿 이미지에 포함)

---

## 슬라이드 2 (노출량 요약 / 공공기관 데이터 기반 통계) 채울 내용

- **헤더**
  - 제목: `{광고주명}` 예상 노출량 리포트
  - 부제: 지하철 역사(역/호선) 유동인구 공개 데이터 기반, 해당 시간대·기간의 광고 배너 예상 노출량 요약
  - 탭: 공공데이터 기반 추정 / 배너 노출량 (선택 탭)
- **하이라이트** (우상단): 피크 시간대(예: 07-09 6.8만명), 2순위(예: 09-12 5.4만명)
- **핵심 지표 4개**
  - 역/호선: `{호선}` `{역명}` (예: 5호선 여의도역)
  - 일평균 유동인구: N.N만명 (공공 데이터 승·하차 기반)
  - 게재 기간: N일 (기간 내 합산 추정)
  - 예상 총 노출량: N.N만명 (시간대별 합산)
- **좌측**: 시간대별 예상 노출량 표 — 시간대, 예상 노출량(만명), 비중(%), 순위, 막대 시각화
- **우측**: 시간대별 노출량 막대그래프, 시간대 비중 도넛차트
- **푸터**: ※ 본 수치는 공공데이터(역사별 승·하차/유동인구) 기반 추정치이며, 실제 광고 노출 수는 편차가 있을 수 있습니다.

## 슬라이드 4~ (촬영 광고 사진) 채울 내용

- **템플릿**: `slide-03-banner.png` (촬영 광고 사진용 지면)
- **배치 규칙**
  - **슬라이드당 2장** 촬영 이미지 배치
  - 촬영 이미지가 **n장**이면, 동일한 템플릿 슬라이드를 **⌈n/2⌉개** 추가 (예: 5장 → 3슬라이드, 6장 → 3슬라이드)
- **채울 내용**: 각 슬라이드의 2개 슬롯에 촬영본 이미지 삽입 (1·2장 → 1슬라이드, 3·4장 → 2슬라이드, …)

## 템플릿 이미지 제출 방법

- **슬라이드 1(표지), 2(요약), 3(시간대별), 촬영용(배너)** 에 쓸 **배경/지면 이미지**를 하나씩 주시면 됩니다.
- **촬영 광고 사진(4~)** 용 템플릿: `slide-03-banner.png` (슬라이드당 2장 배치). “”
제공해 주실 때:
- **슬라이드 번호 + 이름**을 같이 알려주시면 헷갈리지 않게 정리하겠습니다.  
  예: `1 표지`, `2 노출량 요약`, `3 시간대별`, `slide-03-banner 촬영 광고 사진`
- 이미지 파일은 프로젝트 안 **`public/report-ppt-templates/`** 에 저장해 두겠습니다.  
  - `slide-01-cover.png`, `slide-02-summary.png`, `slide-03-timeband.png`, `slide-03-banner.png`, **`slide-04-End.png`**
