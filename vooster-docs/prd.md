## Product Requirements Document (PRD)

### 1. Introduction

*   **Product Name:** 옥외 광고 게재 현황 자동 보고 툴
*   **Purpose:** 옥외 광고 운영 담당자의 광고 게재 현황 보고 업무 자동화
*   **Target User:** 옥외 광고 운영 담당자

### 2. Goals

*   **Business Goals:**
    *   옥외 광고 운영 서비스의 경쟁력 강화
    *   광고 운영 비용 절감
*   **User Goals:**
    *   광고 게재 현황 보고 업무 시간 단축
    *   정확하고 신속한 보고
    *   업무 효율성 향상

### 3. Features

*   **Core Features:**
    *   **광고 촬영:** 스마트폰 카메라를 이용하여 옥외 광고 촬영
    *   **광고주 자동 인식:** 촬영된 광고 이미지에서 광고주 자동 인식
    *   **위치 정보 기록:** 촬영된 광고 위치를 GPS 기반으로 자동 기록
    *   **이메일 자동 발송:** 촬영된 광고 이미지와 위치 정보를 광고주 담당자에게 이메일로 자동 발송
*   **Additional Features:**
    *   OCR (광학 문자 인식) 기능을 통해 이미지에서 텍스트 추출 및 광고 내용 자동 분석

### 4. User Stories

*   **As a** 옥외 광고 운영 담당자,
*   **I want to** 스마트폰으로 광고를 촬영하고 광고주를 자동으로 인식하고 싶다.
*   **So that** 광고 게재 현황 보고 업무 시간을 단축하고 싶다.
*   **As a** 옥외 광고 운영 담당자,
*   **I want to** 촬영된 광고 이미지를 광고주 담당자에게 즉시 이메일로 발송하고 싶다.
*   **So that** 광고주에게 신속하게 광고 게재 현황을 보고하고 만족도를 높이고 싶다.

### 5. Release Criteria

*   광고 촬영 기능이 정상적으로 작동해야 함
*   광고주 자동 인식 기능의 정확도가 95% 이상이어야 함
*   이메일 자동 발송 기능이 안정적으로 작동해야 함
*   GPS 기반 위치 정보 기록 기능이 정확하게 작동해야 함

### 6. Future Considerations

*   광고 효과 분석 기능 추가 (노출 횟수 추정, 시간대별 분석 등)
*   경쟁사 광고 모니터링 기능 추가
*   다양한 광고 형식 지원 (디지털 광고, 영상 광고 등)
