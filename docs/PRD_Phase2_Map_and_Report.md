## 1. 🎯 프로젝트 개요 (Overview)

### 1.1 배경 (Background)

현재 AdMate Vision은 광고 촬영 및 기본 데이터 수집(MVP) 단계에서 안정화되었습니다. 이제 수집된 데이터를 직관적으로 시각화하고, 비즈니스 가치를 입증할 수 있는 보고서 형태로 가공하여 **B2B 고객(광고주, 대행사)에게 실질적인 인사이트를 제공**해야 합니다.

### 1.2 목표 (Objectives)

1.  **데이터 시각화 (Data Visualization)**: 지도(Map) 기반의 대시보드를 통해 광고 분포와 밀집도를 한눈에 파악하게 한다.
2.  **자동화 리포팅 (Automated Reporting)**: 클릭 몇 번으로 AI 분석이 포함된 전문적인 성과 보고서(PDF)를 생성한다.

---

## 2. 🗺️ 기능 1: 스마트 맵 대시보드 (Smart Map Dashboard)

### 2.1 주요 기능 (Key Features)

- **히트맵(Heatmap) 모드**: 광고 촬영 빈도가 높은 지역을 색상(Red > Yellow > Green)으로 시각화.
- **핀(Pin) 클러스터링**: 줌 아웃 시 인접한 마커를 그룹화(예: '강남구 128개')하여 시인성 확보.
- **상세 정보 팝업**: 지도 위 마커 클릭 시 해당 광고의 썸네일, 광고주명, 촬영 일시, AI 신뢰도 표시.
- **필터링**: 지역별(구/동), 광고주별, 날짜별 필터 적용 시 지도 실시간 업데이트.

### 2.2 사용자 흐름 (User Flow)

1.  관리자 대시보드에서 '지도 보기' 탭 클릭.
2.  전체 광고 분포 확인 (줌 인/아웃).
3.  특정 지역(예: 강남구)으로 이동하거나 필터 적용.
4.  개별 마커 클릭 -> 팝업에서 간단 정보 확인 -> '상세 보기' 클릭 시 갤러리 상세 페이지로 이동.

### 2.3 기술적 고려사항 (Tech Specs)

- **Map Library**: `react-leaflet` (오픈소스, 비용 효율적) 또는 `Google Maps API` (고토화 시). 초기엔 Leaflet 권장.
- **Performance**: 대량의 마커 렌더링 시 성능 저하 방지 (Clusterer 사용 필수).

## 3. 📄 기능 2: AI 비즈니스 리포팅 (AI Business Reporting)

### 3.1 주요 기능 (Key Features)

- **원클릭 PDF 생성**: 현재 조회 중인 데이터(또는 선택한 캠페인)를 깔끔한 A4 레이아웃의 PDF로 변환.
- **AI 인사이트 요약**: Gemini Pro를 활용하여 수집된 데이터를 텍스트로 요약 및 제언.
  - _예: "이번 달 삼성동 지역 노출이 전월 대비 15% 증가했으며, 주로 출근 시간대(08~09시)에 집중되었습니다."_
- **주요 지표 시각화**: 보고서 내에 '일별 촬영 추이', '지역별 분포' 차트 자동 삽입.
- **커스터마이징**: 보고서 상단 로고 및 타이틀 변경 기능.

### 3.2 사용자 흐름 (User Flow)

1.  '보고서(Reports)' 페이지 접속.
2.  보고서 생성 조건 설정 (기간, 대상 광고주).
3.  'AI 분석 포함' 옵션 체크 후 '보고서 생성' 클릭.
4.  (로딩: AI 분석 중...)
5.  생성된 리포트 미리보기 확인 -> 'PDF 다운로드' 또는 '이메일 발송'.

### 3.3 기술적 고려사항 (Tech Specs)

- **PDF Generation**: `jspdf` 또는 `react-pdf` 라이브러리 사용.
- **AI Integration**: 기존 Gemini API 연동 활용, 프롬프트 엔지니어링 필요(데이터 요약 전문).

## 5. 🎨 디자인 요구사항 (Design Req)

> **Designer Note**: "프리미엄하면서도 신뢰감을 주는 Data Visualization 디자인이 핵심입니다."

- **Color**: 지도의 포인트 컬러는 브랜드 컬러(`Indigo-500`)를 메인으로 하되, 상태값(정상/오류)은 직관적인 `Emerald`/`Rose` 컬러 사용.
- **Typography**: 보고서는 가독성이 최우선. 명조체보다는 깔끔한 고딕(Pretendard) 유지.
- **Interaction**: 마커 호버/클릭 시 부드러운 애니메이션 적용.
