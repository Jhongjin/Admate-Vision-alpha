## 1. 서비스 개요

### 옥외 광고의 모든 순간, AdMate Vision으로 완성하다

- **Instant Capture**: 촬영 즉시 역명/광고주 자동 인식
- **Smart Matching**: AI 기반 정확한 브랜드 매칭
- **Auto Reporting**: 유동인구 데이터 기반 프리미엄 리포트 발송

## 2. 시작하기

### 📲 앱 설치로 더 편하게

1. 모바일 브라우저(Chrome/Safari) 접속
2. **[홈 화면에 추가]** 버튼 클릭
3. 앱 아이콘으로 전체 화면 실행

> 💡 **Tip**: 별도 비밀번호 없이 이메일만으로 3초 만에 로그인 가능!

## 4. 촬영 및 보고 (Capture Flow)

### STEP 1: 촬영 (Capture)

![bg left:35% 80%](images/capture.png)

1. **[촬영 시작하기]** 터치
2. 광고판이 꽉 차게 구도 잡기
3. **위치 권한 필수 허용** (역명 인식)

---

### STEP 2: 확인 및 설정 (Confirm)

**AI 분석 결과 확인 & 발송 옵션**

1. **파일명 수정**: `역명_광고주_호선` 자동 생성
2. **수신자 선택**:
   - `광고주 담당자`: 내부 담당자 자동 참조(CC)
   - `캠페인 담당자`: 내부 검토용
3. **게재 기간**: PDF 리포트 표기용

---

### STEP 3: 리포트 발송 (Report)

**데이터 기반 Premium Report**

- **AdMate Score**: 공공데이터(유동인구) 기반 성과 점수
- **One-Touch 발송**: 버튼 하나로 PDF 생성 및 이메일 전송 완료

---

## 5. 광고주 관리 (Advertisers)

**AI 인식률을 높이는 핵심**

![bg right:40% 90%](images/advertisers.png)

- **프로필 아바타**: 브랜드 이니셜로 시각적 구분
- **매칭 팁 (옵션)**:
  - 브랜드의 영문명, 약어, 별칭 입력
  - 예: `토스`, `Toss`, `Viva Republica`
- **벌크 업로드**: CSV로 수백 개 광고주 일괄 등록

---

## 6. 보고 목록 (Reports) & 계정

**체계적인 이력 관리**

![bg left:35% 80%](images/reports.png)

- **썸네일 리스트**: 사진으로 보고서 식별
- **상태 배지**: 발송/미발송 상태 확인
- **재발송**: 클릭 시 상세 화면에서 재전송 가능
