# PPT 노출량 보고서 템플릿

## 1) .pptx 템플릿 사용 (권장 — 덮어씌우기 없음)

**이 폴더에 `report_template.pptx` 를 두면**, 코드가 그 파일을 **그대로 열어서** `{placeholder}` 만 실제 값으로 **치환**합니다.  
배경 이미지 위에 글씨를 덮어 쓰지 않고, **템플릿 디자인을 유지한 채 값만 넣는 방식**입니다.

- **필수 placeholder** (슬라이드/텍스트·표 셀 안에 넣어 두세요):
  - `{광고주명}` → 광고주명
  - `{년.월.일}` → 보고 일자 (예: 2026.02.06)
  - `{캠페인 담당자 이름}`, `{캠페인 담당자 이메일}` → 문서 끝 슬라이드
  - `{역/호선}` 또는 `{역사명}` + `{n호선}` → 역/호선
  - `{일평균 유동인구}`, `{게재 기간}`, `{예상 총 노출량}` → 노출량 요약
- 슬라이드 구성·디자인은 **전부 템플릿 .pptx 에 맞춰** 두시면 됩니다.

## 2) 이미지 템플릿 (report_template.pptx 없을 때)

**`report_template.pptx` 가 없으면** 아래 이미지 파일을 **배경**으로 깔고, 그 위에 텍스트를 **덮어써** 넣는 방식으로 동작합니다.

| 파일명 | 용도 |
|--------|------|
| `slide-01-cover.png` 또는 `.png.jpg` | 표지 |
| `slide-02-summary.png` 또는 `.png.jpg` | 노출량 요약 |
| `slide-03-banner.png` 또는 `.png.jpg` | 촬영 광고 사진 |
| `slide-04-End.png` 또는 `.png.jpg` | 문서 끝 |

자세한 순서·용도는 `vooster-docs/report-ppt-slide-templates.md` 참고.
